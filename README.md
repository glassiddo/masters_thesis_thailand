# Master's Thesis: Open-Burning Ban in Northern Thailand (2013-)

This repository contains my master's thesis (PDF) and supporting code on the effects of the open-burning ban in Northern Thailand that started in 2013.

**Status / disclaimer:** This is not a published work and has not been peer reviewed. All errors are my own.

## Contents

- `paper.pdf`: the thesis document
- `do/`: analysis code and scripts

## Data Availability

The data needed to fully reproduce the analysis is **not included** in this repository. Some inputs are not publicly redistributable and/or require requesting access (e.g., FIRMS).

If you would like access instructions or have questions, please open a GitHub issue in this repository (I prefer GitHub over email for communication).

`slides.pdf` is intentionally not included in the public repository due to potential third-party image licensing constraints.
